## To run

# Using executable file provided (from */dist*)
Run `./main`.

# Using python3 (from */src*)
Make sure you have installed all the required libraries.

Run `python3 src/main.py`.

## Libraries used
- PySimpleGUI
- Tkinter (tk) - PySimpleGUI depends on it
- ClipsPy
